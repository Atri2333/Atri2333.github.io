然而，对于对象自己实现的方法，接口没办法引用。

实际上，在我整的JAVA速成Day1那篇介绍集合的文章中，我留了个坑：用Map引用TreeMap。

实际上这里的Map，以及List，都是接口。

它们引用的对象，只能调用自己即继承的接口内部的方法。

因此对于TreeMap中独属于该类的方法，接口Map不能调用。因此你会发现IDEA的自动补全里少了点API。
